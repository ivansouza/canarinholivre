# 🐦 Canarinho Livre

**Camiseta open source inspirada na seleção brasileira.**

O Canarinho Livre é um projeto totalmente livre — qualquer pessoa pode usar, modificar, produzir e comercializar o design sem restrições de direitos autorais ou medo de falsificação.

## 📋 Sobre o Projeto

- **Design:** Camiseta amarela com escudo verde no lado esquerdo do peito
- **Símbolo:** Canarinho amarelo estilizado, peito estufado, olhando para a direita
- **Estrelas:** 5 estrelas acima do escudo
- **Licença:** Nenhuma — o design é **domínio público**. Use, venda, modifique à vontade.

O futebol é do povo. A camiseta também deveria ser.

Sem registro de desenho industrial. Sem direitos autorais sobre o visual. A marca "Canarinho Livre" pode ser usada livremente por qualquer pessoa.
